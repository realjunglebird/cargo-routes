# Cargo Routes

Cargo Routes - это учебный проект по созданию инструмента визуализации графа зависимостей для менеджера пакетов.

## Этапы выполнения задания:
- Этап 1. Минимальный прототип с конфигурацией - Выполнен
- Этап 2. Сбор данных - Выполнен
- Этап 3. Основные операции - Выполнен
- Этап 4. Дополнительные операции - Текущий
- Этап 5. Визуализация

- *Вариант - 24*

## Текущие возможности:
### 1 этап
- Источником настраиваемых параметров является конфигурационный файл формата JSON.
- К настраиваемым параметрам относятся:
  - Имя анализируемого пакета
  - URL-адрес репозитория или путь к файлу тестового репозитория
  - Режим работы с тестовым репозиторием
  - Версия пакета
  - Имя сгенерированного файла с изображением графа
  - Режим вывода зависимостей в формате ASCII-дерева
- Обработка ошибок для всех параметров

### 2 этап
- Используется формат пакетов Rust (Cargo)
- Информация выводится для заданной пользователем версии пакета
- Информация о прямых зависимостях извлекается с использованием URL-адреса 
репозитория или пути к локальному репозиторию 
- На экран выводятся все прямые зависимости заданного пользователем пакета 

### 3 этап
- Граф зависимостей генерируется с помощью алгоритма DFS без рекурсии
- Случаи наличия циклических зависимостей обрабатываются корректно
- Поддерживается режим тестирования: вместо URL реального репозитория можно указать путь к файлу описания графа репозитория, где пакеты называются большими латинскими буквами
